# OG 이미지 전용 폰트

satori는 ttf/otf/woff만 읽으므로 빌드 타임 OG 렌더용 원본 TTF를 여기에 둔다 (웹 폰트는 @fontsource 그대로).

- `GowunBatang-Regular.ttf` / `GowunBatang-Bold.ttf` — 출처: github.com/google/fonts (`ofl/gowunbatang`), SIL Open Font License 1.1
- `Cinzel-Regular.ttf` — 출처: Google Fonts 공식 서빙(fonts.gstatic.com, wght 400 정적 인스턴스 — 저장소 원본 `Cinzel[wght].ttf`는 variable이라 satori의 opentype.js가 파싱하지 못한다), SIL Open Font License 1.1
