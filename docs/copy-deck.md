# 📜 Copy Deck — 방랑자의 어투 (v4)

> 화면에 보이는 모든 UI 문구의 단일 원천. **여기 있는 문장을 그대로 쓴다.**
> 원칙: 방랑자는 정통 하이 판타지의 주민이다. 그는 "클릭"하지 않고 "길을 나선다".
> 단, ⚠️ WORLDBOOK §1.2 절대 규칙 — `<title>`·meta·OG·aria-label·URL은 평범하고 정확하게.
> 어투 헌장: ① 짧고 단단하게(고서는 수다스럽지 않다) ② 명령형 대신 권유형 ③ 이모지 금지, ✦만 허용 ④ 유머는 건조하게 반 스푼.

## 6. 방랑자의 일지

| 위치 | 문구 |
|---|---|
| 배너 플레이버 | 완성되지 않아도 좋다. 오늘 걸은 만큼만 적는다. |
| 월 구분 | {월}월의 길 (예: 칠월의 길) |
| 이달의 기록 위젯 | ✦ 이번 달, {N}일을 기록했다 ✦ |
| 낱장 카운트 | {N}장의 낱장이 쌓였다 · 낱장을 짚으면 그 자리에서 펼쳐진다 |
| 낱장 고유 주소 링크 | 이 낱장만 보기 → |
| 빈 달 | 이 달의 수첩은 비어 있다. 길이 없었던 것은 아니다. |
| 여정의 끝 (첫 낱장 아래) | 여기서 길이 시작되었다. 첫 낱장 이전의 기록은 없다. |

## 7. 여행자의 쉼터

| 위치 | 문구 |
|---|---|
| 배너 플레이버 | 먼 길을 오셨군. 난롯가에 앉으시게. |
| 소개 섹션 제목 | 방랑자에 대하여 |
| 이름 풀이 | 여행자들은 나를 리베리터(Liberiter)라 부른다 — '책(Liber)' + '여정(Iter)'. 지식의 숲을 지나는 긴 여행. *(방랑자는 여관 주인이 아니라 오래 머무는 손님 — 직업 언급 금지)* |
| 인장 플레이트 | SIGILLUM VIATORIS · 방랑자의 인장 |
| 인장 모토 (이탤릭 영문) | *Liberiter — A Grand Journey Through the Forest of Knowledge* |
| 칭호의 방 제목 | 여정의 기록 — 칭호의 방 |
| 취지 섹션 제목 | 이 교차로에 대하여 |
| 취지 카피 | **길 위에서 배운 것은 길 위에 남긴다** — 기록은 미래의 나에게 보내는 전서구다. · **주제마다 건물을 세웠다** — 역사에서 수학까지, 신앙에서 소설까지. 관심사가 여러 갈래라 교차로가 되었다. · **안개는 예고편이다** — 아직 기록이 없는 장소는 안개에 잠겨 있다. 첫 기록이 쓰이는 날, 안개가 걷힌다. · **완성되지 않아도 좋다** — 세 줄의 일지도 당당하게 발행한다. 이 교차로의 적은 완벽주의다. |
| 복사 토스트 (봉인 서신) | 주소를 필사해 두었다 ✦ |
| 연락 섹션 제목 | 전서구를 보내려면 |
| 연락 각주 | 답신은 바람의 사정에 따라 늦을 수 있다. |
